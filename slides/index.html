What is Node

  • For backend development using JavaScript.
  • Collection of libraries written in JavaScript, C and C++.
  • Event-driven, non-blocking I/O model.

What is Node Good For

  • Fast, scalable, data-intensive, realtime apps.
  • Chat apps, analytics, games, etc.
  • nodejs.org/industry
